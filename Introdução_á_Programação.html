Disciplina: Introdução à Programação

A Introdução à Programação no curso de Jogos Digitais tem como objetivo ensinar os conceitos fundamentais da programação, essenciais para criar a lógica por trás dos jogos. O aluno aprende a desenvolver códigos utilizando linguagens de programação, como C++, Python ou até mesmo C#, que são amplamente usadas na indústria. Além disso, são abordados os princípios básicos de algoritmos, estruturas de dados, controle de fluxo e manipulação de variáveis, conceitos que são fundamentais para criar e controlar a dinâmica dos jogos, como movimentação de personagens, interação do jogador e gerenciamento de eventos.
